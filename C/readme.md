
# 📘 C 언어 산술 연산 프로그램 설명

이 문서는 두 정수를 입력받아 산술 연산(덧셈, 뺄셈, 곱셈, 나눗셈)을 수행하고,  
각 연산을 함수로 구현한 C 프로그램에 대한 설명입니다.

---

## 🧩 프로그램 구조


---

## 📂 주요 파일 구성

- `main()` 함수  
  사용자로부터 두 정수를 입력받고, 산술 연산 결과를 출력합니다.

- `add(int a, int b)`  
  두 수의 덧셈 결과를 반환합니다.

- `subtract(int a, int b)`  
  두 수의 뺄셈 결과를 반환합니다.

- `multiply(int a, int b)`  
  두 수의 곱셈 결과를 반환합니다.

- `divide(int a, int b)`  
  두 수의 나눗셈 결과를 `float` 형태로 반환합니다.  
  0으로 나누는 경우는 `main()`에서 별도로 처리합니다.

- `test_operations()`  
  각 연산 함수가 올바른 결과를 출력하는지 간단한 테스트를 수행합니다.
